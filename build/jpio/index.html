Raspberry Pi based analog/digital input/output to OSC interface.

This interface is designed to be reliable and flexible. Based on the Automation Hat, it can interface digital and analog inputs and outputs with OSC messaging. The operating system runs in RAM off of a read only disk to minimize the possibility of disk corruption. It can run off of POE, has automatic cooling and an integrated display. Assembled with custom laser cut and 3D printed components.
